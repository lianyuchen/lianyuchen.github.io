从头搭建环境变量

java 开发环境

背景：由于公司刚给配的iMac，以致于要在新的电脑上搭建开发环境。下面来描述下从头到尾搭建开发环境的全过程。

由于我是从事Android开发，因此本篇文章就主要描述搭建Android开发相关的环境，后续会补充上有关mysql，web方面有关的环境搭建。

准备工作（工具、软件）：

  1. jdk
  2. Android SDK
  3. Android Studio
  4. Cornerstone(包管理器)
  5. Charles(抓包工具)
  6. Google Chrome（这个浏览器非常重要，通过google账号，可以同步所有的平时用到的书签）
  7. Git（不用说大家应该也知道，版本管理工具）
  8. oh-my-zsh(用来提升mac终端的逼格＝＝)
  9. SubLime Text（可以支持多种语言的文本编辑器）

软件的下载地址：

  1. [jdk下载地址][1]
  2. [Android SDK和Studio下载地址][2]

这里就给出以上两个下载地址，其他程序在网上都找得到，实在找不到的call me。

安装过程

一. 安装java，只需点击jdk的dmg安装程序，按照提示进行下一步即可。
二. 安装Android SDK，将下载好的SDK安装包解压到你指定的文件夹（/User/Download/software/）,此时你的sdk路径应该是（/User/Download/software/{sdk_path}），将sdk路径下的tools和platform-tools配置到PATH环境变量中。

三. 打开Android Studio的安装包，进行安装。安装结束后，打开Android Studio，程序会指引做一下配置操作，

四. 安装oh-my-zsh。oh-my-zsh的github，https://github.com/robbyrussell/oh-my-zsh，在终端中输入

便会自动下载安装zsh，在此期间，如果电脑没有安装git，则会提示你安装git。
五. 安装包管理器等等软件。

结束语

此次重新搭建开发环境大约花费了我5个小时的时间，不过其中却不仅仅只有上述内容中的东西，仅此记录一下本次的搭建环境过程，为以后也许还会碰到类似的情况做个奠基。
